# Zeerostock Developer Assignment

This repository contains both developer assignments for Zeerostock:

| Part | Assignment | Port |
|------|-----------|------|
| **Part A** | Inventory Search API + UI | 3000 |
| **Part B** | Inventory Database + APIs | 4000 |

---

## Part A — Quick Start

```bash
cd part-a/backend
npm install
npm start
# Open http://localhost:3000
```

## Part B — Quick Start

```bash
cd part-b
npm install
npm start
# API at http://localhost:4000
```

See each folder's own `README.md` for full API docs, schema details, and design decisions.
